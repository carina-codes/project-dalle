*** FRONTEND:

JSX stands for JavaScript XML. 
JS is standard javascript, JSX is an HTML-like syntax that you can use with React to (theoretically) make it easier and more intuitive to create React components.
JSX performs optimization while compiling the source code to JavaScript.
The generated code runs faster than an equivalent code written directly in JavaScript.
